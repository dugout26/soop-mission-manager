# ⚔ SOOP 대결미션 매니저

SOOP 방송의 대결미션 별풍선을 **실시간 자동 감지**하여 미션을 관리하는 도구입니다.

## ✨ 주요 기능

- ⭐ **별풍선 자동 감지** — 500개, 1000개 등 기준에 맞는 별풍선 자동 미션 등록
- 🔗 **역팬 링크 자동 생성** — 클릭 한 번으로 해당 유저 채널 이동
- 📸 **방셀포즈 기록** — 원하는 포즈 내용 메모
- ✅ **완료 체크** — 판매종료 스탬프
- 📊 **실시간 대시보드** — 브라우저에서 관리
- 🔔 **알림음** — 미션 등록 시 효과음

## 🚀 설치 방법

### 1. Node.js 설치
https://nodejs.org 에서 LTS 버전 다운로드 & 설치

### 2. 프로젝트 폴더에서 패키지 설치
```bash
npm install
```

### 3. 설정
`.env` 파일을 열어서 스트리머 아이디를 입력하세요:
```
STREAMER_ID=스트리머아이디
```

또는 대시보드에서 직접 입력해도 됩니다.

### 4. 실행
**Windows:** `start.bat` 더블클릭
**Mac:** `sh start.sh` 또는 터미널에서:
```bash
node server.js
```

### 5. 대시보드 열기
브라우저에서 http://localhost:3000 접속

## 📋 사용법

1. 스트리머 ID 입력 후 "연결" 클릭
2. 방송 중 별풍선이 감지되면 자동으로 미션 등록됨
3. 역팬 미션: "역팬 쏘기→" 버튼 클릭하면 해당 유저 채널로 이동
4. 방셀포즈 미션: 포즈 입력란에 원하는 포즈 내용 입력
5. 완료 시 "✅ 완료" 버튼 클릭

## ⚙ 미션 기준 변경

`server.js`의 `CONFIG.MISSION_THRESHOLDS`를 수정하세요:

```javascript
MISSION_THRESHOLDS: [
  { count: 500, type: 'fan', label: '역팬' },
  { count: 1000, type: 'pose', label: '방셀포즈' },
  { count: 2000, type: 'custom', label: '특별미션' }, // 추가 가능
],
```

## ⚠ 참고사항

- 이 도구는 `soop-extension` 비공식 라이브러리를 사용합니다
- **방송 중**에만 채팅 서버에 연결 가능합니다 (방송이 꺼지면 자동 재연결 대기)
- SOOP 로그인 없이도 별풍선 감지는 가능합니다
- 데이터는 서버 메모리에만 저장되며, 서버 재시작 시 초기화됩니다
